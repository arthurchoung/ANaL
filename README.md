# ANaL

Amiga, NeXTSTEP, and Linux -- Together at last!

## How to compile and run

$ sh makeUtils.sh

$ perl build.pl

To run the window manager:

$ ./anal runWindowManager

To run the iPod style interface:

$ ./anal
